# Desafio 01

### Sobre o desafio

Nesse desafio você desenvolverá uma API para realizar o CRUD de suas *tasks* (tarefas).

Rotas:

- [X] `POST - /tasks`
- [X] `GET - /tasks`
- [X] `PUT - /tasks/:id`
- [X] `DELETE - /tasks/:id`
- [X] `PATCH - /tasks/:id/complete`
